Учебный проект.

Представляет собой сервис для создания и удаления статей, пользователей, лайков в БД MySql.
Работает через терминал. Разработано без использования фреймворков.

Реализовано:
- Маршрутизация
- Классы Request, Response
- DI-контейнер
- Аутентификация
- Логирование
- Команды
- Unit Тесты
